# Azure Enterprise Platform

## Overview

This project demonstrates enterprise-grade Azure infrastructure deployment using modern Infrastructure as Code (IaC) practices.

The platform is built using:

- Terraform
- Ansible
- GitHub Actions
- Azure
- Windows Server
- Kubernetes (AKS)

The goal is to simulate a production-ready enterprise environment following Microsoft Azure best practices.
